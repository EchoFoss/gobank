# Rotas da api

## Create account

**Method: Post**

**Endpoint: /accounts**

## Get all accounts

**Method: Get**

**Endpoint: /accounts**

## Get account by Id

**Method: Get**

**Endpoint: /accounts/{id:int}**

**Header:** 
- {x-jwt-token}: token retornado ao fazer o Login

## Delete account

**Method: Delete**

**Endpoint: /accounts/{id:int}**
